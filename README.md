# T2_NLP

Парсинг и сентимент-анализ отзывов на Google Play для приложения Tele2.

## 📋 Описание проекта

Проект представляет собой анализ тональности и тематическое моделирование отзывов пользователей мобильного приложения Tele2 из Google Play. Используется модель RuRoBERTa для получения эмбеддингов текстов и модель для классификации тональности на русском языке.

## 📁 Структура проекта

```
T2_NLP/
├── NLP (1).ipynb              # Jupyter Notebook с кодом анализа
├── tele2_reviews (1).csv      # Датасет с отзывами
├── Команда Бабки и бабки.pdf  # Документация проекта
└── README.md                  # Этот файл
```

## 📊 Данные

Файл `tele2_reviews (1).csv` содержит следующие колонки:
- **Дата** — дата отзыва
- **Оценка** — рейтинг от 1 до 5
- **Отзыв** — текст отзыва пользователя
- **Имя** — имя автора отзыва
- **Ответ разработчика** — ответ от команды приложения (если есть)
- **Версия приложения** — версия приложения, на которую оставлен отзыв

## 🛠 Используемые технологии и библиотеки

### Основные библиотеки:
- **transformers** — работа с предобученными моделями NLP
- **torch** — фреймворк для глубокого обучения
- **pandas** — обработка данных
- **numpy** — численные вычисления
- **scikit-learn** — машинное обучение (кластеризация, PCA, TF-IDF)
- **umap-learn** — уменьшение размерности для визуализации
- **plotly, matplotlib, seaborn** — визуализация данных

### Модели:
- **ai-forever/ruRoBERTa-large** — получение векторных представлений текстов на русском языке
- **blanchefort/rubert-base-cased-sentiment** — классификация тональности (позитивный/негативный/нейтральный)

## 📈 Функционал

Проект выполняет следующий анализ:

1. **Сентимент-анализ** — определение тональности каждого отзыва (позитивный, негативный, нейтральный)
2. **Векторизация текстов** — получение эмбеддингов с помощью RuRoBERTa
3. **Кластеризация отзывов** — группировка похожих отзывов для выявления основных тем
4. **Визуализация** — построение графиков распределения оценок, тональности и кластеров

## 📝 Пример использования

```python
# Загрузка модели для сентимент-анализа
sentiment_pipeline = pipeline(
    "sentiment-analysis",
    model="blanchefort/rubert-base-cased-sentiment",
    tokenizer="blanchefort/rubert-base-cased-sentiment"
)

# Анализ тональности отзыва
result = sentiment_pipeline("Отличное приложение, всё работает!")
print(result)  # [{'label': 'POSITIVE', 'score': 0.98}]
```
